This page contains info about some linux utilities

Topics

  • Managing archives
  • Compressing files
  • Locating files

tar

  • Extract the contents of a .tar.gz package:
   tar xvfz file.tar.gz
  • Create a .tar.gz package with the content of a given directory:
   tar cvfz outfile.tar.gz dirname

zip

  • Example: To compress files using gzip use
gzip filename
  • Example: To uncompress .gz files use
gunzip filename.gz
  • Example: To compress files using zip use
zip out.zip file1.dat file2.dat file3.dat
  • Example: To uncompress .zip files use
unzip file.zip
  • Example: To uncompress .7z files use
p7zip -d file.7z

find

  • Example: To find ‘filename’ in all order-1 subdirectories of /pathdirectory/, use
find /pathdirectory/ -maxdepth 1 -name filename
  • Example: To find all C headers (.h) in all order-3 subdirectories of currect directory, use
find ./ -maxdepth 3 -name '*.h'
